# NSC114 EEG Small-Sample Binary Classification — Code

Analysis code for a small-sample (n=114) EEG + image fusion binary-classification study
(patient-aware out-of-fold validation, restricted parameter search, tail statistics, multi-seed bagging).

> **Data availability:** This repository contains **code only**. No participant data, EEG
> recordings, or identifiers are included or tracked (see `.gitignore`). De-identified data
> are subject to ethics-approval restrictions; see the manuscript's Data availability statement.

See `README_REPRODUCE.md` for the pipeline and `CHANGELOG_fft_fix_20260617.md` for the
FFT frequency-axis fix (2026-06-17).
